# Zendesk Callange

## Installing Prerequisites

Install Node.js and npm

Then, in the checked out git repo run:

```
npm install
```

## Building

```
npm run build
```

## Running

```
npm start
```
or
```
node .
```

## Running Tests

```
npm test
```

Testing with coverage:
```
npm run coverage
```
Results will be viewable with coverage/lcov-report/index.html
