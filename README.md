# Zendesk Callange

## Pre-requisites

* Node.js
* npm

## Installing Prerequisites

```
npm install
```

## Building

```
npm run build
```

## Running

```
npm start
```
or
```
node .
```

## Running Tests

```
npm test
```

Testing with coverage:
```
npm run coverage
```
Results will be viewable with coverage/lcov-report/index.html
